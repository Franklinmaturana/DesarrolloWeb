Primero

Primero debemos crear una cuenta en Github y descargar Git
al tener la cuenta creada, creamos un repositorio.
Le damos un nombre, una descripcion, seleccionamos si el proyecto sera publico o privado, y dejamos sin marcar el check
de crear el README.Le damos a crear repositorio y con eso ya tenemos el repositorio donde alojaremos nuestro proyecto.

Posteriormente desde la terminal
Debemos hacer una configuracion base de git con los siguientes comandos:
  • git config --global user.name "Tu nombre"
  • git config --global user.email "Tu correo de github"

Luego vamos a la carpeta donde esta nuestro proyecto
y ejecutamos los siguientes comandos
  • git init
  • git add .
  • git commit -m "Comentario"
  • git remote add origin https://github.com/NOMBRE_USUARIO/NOMBRE_PROYECTO.git
  • li>git push -u origin master

Como clonar repositorios

Para clonar un repositorio hacemos uso del siguiente comando:

  • git clone linkdelrepositorio
    • ejemplo:
  • git clone https://github.com/Phoenix-encoder/DesarrolloWeb
